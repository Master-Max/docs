***************
CommerceML (1С)
***************

.. important:: 
  
    Статья актуальна для CS-Cart 4.12.2 и более новых версий.

Модуль предназначен для обмена данными с системами учета: 1С, МойСклад, Class365.

Основной упор документация будет делать на описание особенностей реализации обмена данными. Для получения общей информации о формате CommerceML и о протоколе взаимодействия воспользуйтесь официальной документацией от 1С:

* `Описание формата и структуры документов <http://v8.1c.ru/edi/edi_stnd/90/92.htm>`_
* `Описание протокола обмена с сайтом <https://v8.1c.ru/tekhnologii/obmen-dannymi-i-integratsiya/standarty-i-formaty/protokol-obmena-s-saytom/>`_
