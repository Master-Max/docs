************************
Конструктор прайс-листов
************************

Данный модуль позволяет администраторам магазинов создавать и экспортировать любое количество настраиваемых прайс-листов в различных форматах. Эта функция полезна при работе с веб-каталогами товаров, такими как Google Product Search, Shopzilla, Shopping.com и т.д., так как с её помощью вы можете настроить набор полей и их порядок в соответствии с требуемым форматом экспорта.

Пользовательские прайс-листы можно создавать и автоматически загружать на FTP-сервер или в веб-директорию. Для управления этими процессами используется cron-скрипт. Вы можете установить пароль для доступа к cron скрипту в настройках модуля :doc:`/user_guide/addons/data_feeds/set_datafeeds`.

.. note::

    В CS-Cart можно создавать отдельные прайс-листы для каждого магазина. Для этого сначала переключитесь в режим администратора магазина, а затем приступайте к созданию прайс-листа.
